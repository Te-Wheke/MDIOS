# MDIOS

Lightweight multi-operating-system boot/TUI prototype.

## Purpose

This repo contains boot-screen assets, ASCII/TUI artwork, animation experiments, build scripts, and Codex guidance for the MDIOS prototype.

## Structure

- `assets/images/` — source and generated image assets.
- `assets/ascii/` — ASCII artwork and terminal-ready text assets.
- `boot/` — boot animation outputs, storyboards, and layout references.
- `scripts/` — build tools, conversion tools, and examples.
- `docs/` — Codex instructions, guides, and project notes.
- `builds/` — generated boot/build outputs.
- `tmp/` — temporary or unclassified files.

## Current focus

Keep boot/TUI work small, stable, terminal-safe, and testable.
